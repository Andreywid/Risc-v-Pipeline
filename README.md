# Infraestrutura de Hardware - Projeto RISC-V Pipeline 🚀

Este repositório contém a nossa implementação do projeto da disciplina Infraestrutura de Hardware (IF674) no CIn-UFPE. O objetivo do projeto é implementar instruções em um processador RISC-V usando SystemVerilog.
